ABM — the memory that signs a contract

A deterministic memory runtime for compiled knowledge. Unlike vector memories, it predicts in advance how accurately it can answer under a fixed memory budget. Everything below runs in your browser — real XOR binding, real majority-vote trace, real formulas. 90 seconds.

1 · DOCUMENT Your knowledge, as text

2 · COMPILE Facts an LLM (or a regex) extracted

3 · MEMORY One 2048-bit trace holds everything

Every fact is XOR-bound and majority-voted into this single bit pattern — …

4 · ASK Multi-hop questions, answered by algebra

5 · VERIFY The contract, tested live

The panel on the right was computed, not measured. Press verify: every stored fact is queried back and the measured accuracy is compared with the promise.
