## Summary

Defines the interface for all BSM encoders.

## Specification

### Protocol

```python
class BSMEncoder(Protocol):
    state_dim: int

    def encode(self, data: Any) -> np.ndarray:
        """Returns (D,) or (N, D) int8 array in {-1, +1}."""
        ...
```

### Requirements

1. `encode()` MUST return `np.ndarray` with dtype `np.int8`.
2. Values MUST be in {-1, +1}.
3. Single input returns shape (D,); batch returns (N, D).
4. Encoder MUST be deterministic for given (seed, implementation_version).
5. `state_dim` MUST match the encoder's output dimension.

### Registration

```python
from bsm.encoder import register_encoder

@register_encoder("name")
class MyEncoder:
    ...
```

## Rationale

A protocol (not a base class) allows maximum flexibility.
Custom encoders can wrap any feature extraction pipeline.

## Compatibility

All built-in encoders (Hash, Projection, Learned) conform to this protocol.
