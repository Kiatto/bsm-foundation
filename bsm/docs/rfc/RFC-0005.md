## Summary

Specifies the sleep cycle for memory consolidation and forgetting.

## Specification

### Algorithm

```
Input:  Memory M = {(S_i, P_i, m_i)}
        Parameters: forget_threshold, max_age, min_access_rate

1. For each entry i:
     age = now - m_i.ts
     access_rate = m_i.access / max(age, epsilon)

     keep = (m_i.value > forget_threshold)
            OR (age < max_age)
            OR (access_rate > min_access_rate)

2. Remove entries where keep = False.
3. Normalize remaining values: m_i.value /= max(m_i.value)
4. Rebuild any hash indices.
5. Return count of forgotten entries.
```

### Default Parameters

| Parameter | Default | Description |
|-----------|---------|-------------|
| forget_threshold | 0.3 | Minimum value to survive |
| max_age | 3600 s | Maximum age without access |
| min_access_rate | 0.001 / s | Minimum access frequency |

### Rationale

Sleep is inspired by biological memory consolidation.
It prevents unbounded growth and maintains retrieval quality.

## Compatibility

Sleep operates only on metadata.  All states and payloads are preserved
for entries that pass the keep predicate.
